# Amigo Secreto

## Descripción
Este proyecto es una aplicación web para realizar un sorteo de "Amigo Secreto". 
En este desafío,  permite a los usuarios ingresar nombres de amigos en una lista 
para luego realizar un sorteo aleatorio y determinar quién es el "amigo secreto".

## Tecnologías
- HTML5
- CSS3
- JavaScript 

## Instalación
- Paso 1: Clonar el repositorio desde GitHub.
- Paso 2: Abrir el proyecto.  

## Uso
1. Escribe el nombre de un amigo en el campo de texto.
2. Haz clic en el botón **Añadir** para agregarlo a la lista de amigos.
3. Repite el proceso para todos los amigos que quieras incluir.
4. Una vez agregados todos los nombres, haz clic en **Sortear Amigo**.
5. El nombre del amigo secreto aparecerá en la sección de resultados.

## Funcionalidades principales
- Agregar amigos mediante el campo de texto.
- Validación de entrada: no se permiten campos vacíos.
- Lista dinámica de amigos añadidos.
- Sorteo aleatorio de un amigo secreto.
